# Repository agent instructions

This repository contains portable Agent Skills for auditing and reducing
generic, unsupported, or unnecessary design output.

When editing:

1. Preserve open Agent Skills compatibility.
2. Do not introduce vendor-only frontmatter into core skills.
3. Keep deterministic scripts non-interactive and network-free.
4. Never claim that an artifact was AI-generated solely from aesthetic signals.
5. Preserve accessibility criteria and evidence labels.
6. Update `manifest.json`, evals, changelog, and tests when adding a skill.
7. Run repository validation before finalizing.
8. Do not rewrite user-provided facts or evidence as invented marketing claims.
